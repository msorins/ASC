DOSBox

http://www.dosbox.com/ (very easy to port to different platforms)

The page where it can be downloaded from: http://www.dosbox.com/download.php?main=1

In order to use it, we have to mount in the emulator the needed folders from the harddisk.
Exemple:
    > Mount C C:/ASC
    > C:
 if we want to work in C:\ASC.
